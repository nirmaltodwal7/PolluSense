# PolluSense 🌫️ — Hybrid AI Air Quality Prediction System

A full-stack air quality monitoring and prediction system using a **Random Forest + LSTM hybrid model**, Flask backend, and React dashboard.

## 🤖 Hybrid ML Model

```
Raw Data → MinMaxScaler
    ↓
Random Forest Regressor
  • Trains on all pollutant features
  • Identifies top important features (PM2.5, PM10, NO2, CO)
  • Outputs RF Baseline AQI prediction
    ↓
LSTM Neural Network
  • Input: 60-day sliding window sequences
  • Architecture: LSTM(128) → Dropout → LSTM(64) → Dense(1)
  • Captures time-series patterns
    ↓
Ensemble (40% RF + 60% LSTM) → Final AQI
```

## ⚙️ Setup & Run

### Prerequisites
- Python 3.9+
- Node.js 18+

### 1. Install backend dependencies
```bash
cd backend
pip install -r requirements.txt
pip install tensorflow
```

### 2. Generate synthetic dataset
```bash
python generate_dataset.py
```

### 3. Train models (one-time)
```bash
python train_random_forest.py
python train_lstm.py
```
> LSTM training takes ~3–5 minutes on CPU. Early stopping is configured.

### 4. Start Flask API
```bash
python app.py
# → http://localhost:5000
```

### 5. Start React frontend (new terminal)
```bash
cd pollusense-ui
npm install
npm run dev
# → http://localhost:5173
```

## 🌐 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/health` | Health check |
| `GET` | `/cities` | List of supported cities |
| `POST` | `/predict` | 1–7 day AQI forecast |
| `POST` | `/predict/single` | Single date prediction |

### Example Request
```json
POST /predict
{
  "city": "Delhi",
  "days": 7
}
```

### Example Response
```json
{
  "city": "Delhi",
  "predictions": [253.8, 249.7, 244.2, 238.0, 224.0, 202.0, 170.0],
  "categories": ["Very Unhealthy", "Very Unhealthy", "...", "Unhealthy"],
  "dates": ["2026-04-20", "2026-04-21", "..."],
  "alert": "⚠️ VERY UNHEALTHY: Health alert — everyone may experience serious health effects.",
  "forecast": [...]
}
```
